# Code of Conduct

## The short version

Be honest. Be respectful. Assume good faith. This project exists to serve people, not platforms or profit. Treat others the way the protocol itself is designed to work: as equals, face to face, with nothing hidden.

## Standards

We welcome anyone regardless of background, experience level, identity, or geography. The following behaviors are expected:

- Honest, constructive communication
- Respect for differing perspectives and experience levels
- Graceful acceptance of constructive feedback
- Focus on what benefits the project and its users

The following behaviors are unacceptable:

- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information without consent
- Any conduct that would be inappropriate in a professional setting

## Scope

This code of conduct applies to all project spaces including issues, pull requests, discussions, and any communication related to the project.

## Enforcement

Instances of unacceptable behavior can be reported to hello@humanexchangeprotocol.org. All reports will be reviewed and responded to. Maintainers have the right to remove, edit, or reject contributions that violate this code of conduct.
